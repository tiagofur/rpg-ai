## Stack Tecnológico

### Tecnologías Usadas

En el desarrollo de este proyecto, hemos optado por React Native y Expo como nuestras tecnologías principales. Esta decisión se tomó debido a la capacidad de React Native para crear interfaces de usuario de alta calidad y la facilidad de uso de Expo para la gestión de entornos de desarrollo y despliegue.

#### Ventajas de React Native + Expo
- **Desarrollo Rápido:** La capacidad de utilizar componentes predefinidos y la configuración mínima requeridas por Expo permiten un desarrollo rápido y eficiente.
- **Multiplataforma:** Con React Native, podemos desplegar nuestra aplicación tanto en iOS como en Android desde una única base de código.
- **Experiencia de Usuario Premium:** Las capacidades nativas de React Native ayudan a lograr un rendimiento óptimo y una experiencia de usuario fluida, similar a una aplicación nativa.

Esta elección refuerza nuestro compromiso por crear un producto de alta calidad que busca brindar una experiencia similar a la de un juego premium en dispositivos móviles.
